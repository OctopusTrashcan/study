集合(Set)


Def1:A set is a callection of distinct objects(定義:一個集合是收集了不同的物件)
The objects inside a set A are called elements of A(而在A裡面的物件稱為A的元素)


Ex:A={1,2}


If x is an element of A
We write x∈A(如果X是A的元素，我們寫作x∈A)
If x is not an element of A
We write x∉A(如果X不是A的元素，我們寫作x∉A)


Def2:The set does not contain any element is called the empty set. donoted∅(一個集合沒有包含任何元素被稱為空集合)


Def3:Let A and B be two sets. We say A is subset of B if "every element of A is also an element of B(讓A和B是兩個集合，如果A的元素都是B的元素，我們說A是B的子集合) 


∀x∈A,x∈B 
∀x,x∈A⟹x∈B


If A is a subset of B,write A⊆B
If A⊆B and A≠B，we say A is proper subset of B，donoted A⊊B
ex:{1,2}⊊{1,2,3} {1,2,3}⊈{1,2}
If A is a set,then∅⊆A


Def4:Let A and B be sets
	1)Union A⋃B A⋃B={x:x∈A or x∈B}
	2)Intersection A⋂B={{x:x∈A and x∈B}}
	3)Difference A\B={x:x∈A and x∉B}
